# Fast API 란?

파이썬 3.6 이상에서 작동하는 , 표준 파이썬 타입 힌트를 바탕으로 한 API 서버 웹 프레임워크

릴리즈 노트로는 2019년 초에 발표했다.

Flask 와 비슷한 구조를 가졌으며,기본적으로 비동기식으로 작성되어 있어 I/O 병목현상이 자주 일어나는 웹서버에서 빠른 성능을 보여준다.
